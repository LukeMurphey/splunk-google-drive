================================================
Overview
================================================

This app provides a mechanism for importing and exporting data to/from Google drive.



================================================
Configuring Splunk
================================================

This app exposes a new input type that can be configured in the Splunk Manager. To configure it, create a new input in the Manager under Data inputs È Google Spreadsheet.
